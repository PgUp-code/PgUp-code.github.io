Host Evasions - 主机规避

了解基于主机安全防护背后的技术，并绕过 Windows 操作系统中最常见的安全产品。

📄️ Windows Internals - Windows 内部原理

学习并理解 Windows 在其核心层面的基本运作原理。

📄️ Introduction to Windows API - Windows API简介

学习如何与 Win32 API 交互，并了解其广泛的使用场景。

📄️ Abusing Windows Internals - 滥用 Windows 内部机制

通过运用与工具无关的现代方法，利用Windows内部组件规避常见检测方案

📄️ Introduction to Antivirus - 防病毒简介

📄️ Signature Evasion - 签名规避

📄️ Runtime Detection Evasion - 运行时检测规避

学习如何使用现代的工具无关方法绕过常见的运行时检测措施，例如 AMSI。

📄️ Evading Logging and Monitoring - 绕过日志记录和监控
